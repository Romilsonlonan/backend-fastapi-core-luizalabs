# ARQUITETURA DO PROJETO CBF MANAGER

## Visão Geral

O CBF Manager é uma aplicação web full-stack para gerenciamento de operações da CBF, desenvolvida com foco em boas práticas de código limpo, segurança, observabilidade e escalabilidade.

---

## Stack Tecnológica

**Backend:**  
- FastAPI (Python)
- SQLAlchemy (ORM)
- Alembic (migração de banco)
- Pydantic (validação)
- JWT (autenticação)
- Passlib (hash de senha)
- Poetry (dependências)

**Frontend:**  
- Next.js (React + TypeScript)
- Tailwind CSS (estilização)
- Radix UI (componentes acessíveis)
- React Hook Form (formulários)
- Zod (validação)
